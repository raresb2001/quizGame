# quizGame

This project is a Quiz Game. 

How is it played?

1. The user enters the menu.
<img width="1353" alt="Menu" src="https://github.com/raresb2001/quizGame/assets/111699122/0f10df27-9e4a-4d94-9c34-32b35318af2e">
2. Select the instructions to see how to play the game.
<img width="1352" alt="instructions" src="https://github.com/raresb2001/quizGame/assets/111699122/20fba919-8d91-4e5e-a87c-1dc53a5038e8">
3. Then select play game to start the game
<img width="1354" alt="playgame" src="https://github.com/raresb2001/quizGame/assets/111699122/3ee56525-44b0-4f61-a712-1ca4f5316a80">
4. If you answer correctly to questions will receive 10 points per question, and if wrong answer will end the game.
<img width="1354" alt="true" src="https://github.com/raresb2001/quizGame/assets/111699122/28fd8f39-e35d-41d7-8b6f-e669784eac0d">
5. View high scores
<img width="1352" alt="highscores" src="https://github.com/raresb2001/quizGame/assets/111699122/e2c85dbc-5b74-4496-8816-16549b038479">
6. To get out of the game you have to select "Quit".
<img width="1349" alt="quit" src="https://github.com/raresb2001/quizGame/assets/111699122/2e3bc202-62b8-4108-83af-37a428fe9ea3">
